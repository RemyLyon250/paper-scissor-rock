LET'S PLAY GAME IN CONSOLE

  1. Open a new tab or navigate to the website where you want to access the console.
  2. Right-click anywhere on the webpage and select "Inspect" from the context menu. Alternatively, you can use the keyboard shortcut by pressing Ctrl+Shift+I (Windows/Linux) or Command+Option+I (Mac).
  3. This action will open the Chrome Developer Tools panel. You should see a panel overlaying the webpage.
  4. In the Developer Tools panel, look for the "Console" tab at the top and click on it. This will open the JavaScript console.
